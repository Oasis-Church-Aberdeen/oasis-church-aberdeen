# Build Oasis Church Aberdeen Website

Build a complete, modern church website with all Priority Matrix features (P0-P3), mock content, and contemporary design.

## Scope

### Pages to Build (from Priority Matrix):
**P0 (Critical):**
- Home / Welcome
- About Us  
- Contact
- Sunday Services + What to Expect

**P1 (High):**
- Sermon Archive (with YouTube thumbnails)
- Children & Youth
- Our Beliefs

**P2 (Medium):**
- Connect Groups
- Calendar/Events
- Online Giving
- Facility Hire

**P3 (Differentiators):**
- "I'm New" / Plan Your Visit
- Partner With Us
- Sermon Quizzes
- Curated Podcasts (church favorites)
- Blog/News
- Resource Library

## Technical Stack
- HTML5 + CSS3 (vanilla, no frameworks)
- Modern CSS Grid/Flexbox
- Responsive design (mobile-first)
- Public domain imagery from Unsplash/Source
- Mock data for all dynamic content
- Interactive sermon quiz functionality

## Design Approach
- Clean, modern aesthetic inspired by Connect City Church (user's favorite)
- Warm, welcoming color palette (suggesting: deep teal + coral/orange accents)
- Large hero sections with compelling imagery
- Card-based layout for sermons/events
- Smooth scroll navigation
- Mobile-optimized hamburger menu

## Content Strategy
- Search-based research for Oasis Church details
- Mock sermons (6-8 with realistic titles/series)
- Mock events (next 3 months)
- Mock blog posts (3-4 recent)
- Mock podcast recommendations (church members' favorites)
- Quiz questions for 2-3 sermons
- Realistic placeholder images
