# Oasis Church Aberdeen Website - Project Brief

## Project Overview
Website for Oasis Church Aberdeen, an evangelical church located in Bridge of Don, Aberdeen, Scotland.

## Church Details
- **Name:** Oasis Church Aberdeen
- **Location:** Laverockbraes, Whitestripes Road, Bridge of Don, Aberdeen AB22 8AW
- **Service Times:** Sundays at 10:30am
- **Mission:** A community of real people discovering life, hope, and purpose together

## Target Audience
- First-time visitors exploring church
- Regular attendees looking for information
- Families seeking children/youth programs
- Community members interested in events and facility hire
- Those wanting to access sermon content online

## Website Pages

### Core Pages (Implemented)
| Page | Purpose |
|------|---------|
| `index.html` | Homepage with welcome, latest sermon, events, quick links |
| `new-here.html` | First-time visitor guide, what to expect |
| `about.html` | Church history, vision, leadership |
| `beliefs.html` | Statement of faith, core beliefs |
| `sundays.html` | Service times, what to expect on Sundays |
| `sermons.html` | Sermon archive with video/audio links |
| `connect-groups.html` | Small groups info and sign-up |
| `children-youth.html` | Kids and youth programs |
| `calendar.html` | Upcoming events and church calendar |
| `blog.html` | News, updates, community stories |
| `podcasts.html` | Church podcast content |
| `resources.html` | Links to recommended podcasts and resources |
| `giving.html` | Online donation page |
| `facility-hire.html` | Hall/room booking for events |
| `partner.html` | Volunteer opportunities and partnerships |
| `quizzes.html` | Interactive quizzes linked to sermons |
| `contact.html` | Contact form, location map, social links |

## Technical Stack
- **Structure:** Static HTML5 website
- **Styling:** Custom CSS (`css/styles.css`)
- **Interactivity:** Vanilla JavaScript (`js/main.js`)
- **Responsive:** Mobile-first design with hamburger navigation
- **Hosting:** TBD (likely Netlify or similar static host)

## Design References
Based on competitor analysis of Aberdeen-area church websites:
- **Preferred modern aesthetic:** Connect City Church (clean, contemporary)
- **Content structure reference:** Gilcomston Church (well-organized)
- **Sermon library inspiration:** DCFC (good audio resource organization)
